These trims on screen can be used instead of the hardware trims, or just as a handy display. ||

If the hardware trims and the screen display move in opposite directions, just recalibrate the trims from the Setup menu.||

If the trims move the control surface of the model the wrong way, check the 'R' box to reverse them. ||

If the hardware trim moves the wrong screen trim, then either 'mode' is set wrongly or the connector is not in the right place, or maybe both.|| 

Remember to set the 'Trim multiplier' under System to a value other than zero. Also note that on the System screen is the option to copy trims to all banks. If this is not set, the trim you set will apply only the the current bank.||

Press one hardware trim at a time. Nothing will happen if you press two or more. ||

Once you find the best place for all trims, you may like to transfer the settings to the subtrims. But note that subtrims affect all banks equally.
