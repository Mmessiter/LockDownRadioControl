Set credentials

Saved to NVS and used on next boot. Edit either field and tap save; the current values are pre-filled.

Model name

Used as this receiver's WiFi AP SSID and its .local hostname. Letters, digits and hyphens only; must start with a letter or digit; no spaces; up to 30 characters.

Factory defaults

Use this before passing the receiver to someone else. Wipes the saved WiFi password, model name, bind state, output protocol and any Rotorflight backups. The receiver reboots and comes up as a fresh unit at RXV2.local.
