
# LockDownRadioControl
This is a 16 channel Radio Control system for models of all kinds. I started this project during Lockdown (May 2020) because I've always loved flying model aircraft - and for a while we all had to stay at home!. 

Back then, I didn't expect it to become useful for flying expensive models. But after flying all my models with it for about 4 years, without even one failure, I am delighted to say it's now proven to be at least as good as, and maybe better than, commercially available systems. 

It is also much cheaper than an equivalent commercial system. I estimate the cost to be around £120 per transmitter, and around £35 per receiver depending on where you buy the components.

# Features 
New features are added from time to time, especially when requested by users. 

Here is a brief summary of the features this radio offers at the time of writing (August 2024).

1. The microcontroller is an **ARM Cortex M7 (at 600 Mhz)**. These are on the Teensy 4.1 and 4.0 boards.
2. The recommended transceivers are Ebyte's **ML01DP5** for the transmitter, and two **ML01SP4**s for the receiver. 
3. 16 Channels.
4. Range of over 1.5 kilometers.
4. Easy to use capacitive touch screen graphic user interface.
4. 32 mixes which can mix inputs or outputs.
5. 4 flight modes (or banks).
6. 90 model memories.
7. Model memory backup and restore using internal SD card.
8. FHSS at 50Hz on the 2.4 Ghz waveband.
9. "AMS" (**A**utomatic **M**odel **S**election) loads correct model memory automatically on connection to model. 
10. Wireless Buddy Box
11. Using AMS, Buddy always loads the same model as Master has loaded, automatically.
12. Wireless model memory exchange. This allows sharing model memories without removing SD card.
13. Telemetry including model volts (and GPS, temperature & barometric pressure, when using custom sensor hub)
14. Speaker gives critical voice messages and other audio prompts.
15. User defined Channel names.
16. User mapped input and output channels.
17. 5 point editable curves for every channel and every flight mode. Supports straight, smoothed, and expo.
18. Failsafe.
19. User definalble switches.
20. Digital trims, rates, subtrim, servo reverse, servo speeds, macros, timer, log files, etc.
21. Support for SBUS, PWM (with definable frequency and centre posiion), and PPM.
