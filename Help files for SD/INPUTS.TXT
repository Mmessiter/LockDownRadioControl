This screen has two main functions:||

(1) An output channel can be mapped from any input device.|
(2) The name of each output channel can be edited.||


One input device can have multiple outputs. Traditional helicopters use input 3 ('Throttle') for both throttle and collective pitch. Although these share an input stick, each still uses its own independent output curves. ||

Inputs 9,10,11 and 12 (and switches) offer three positions, which can be edited from their curve.||

Inputs 13,14,15 and 16 are always centred. Nonetheless these last four channels are useful for mixes and extra functions. Their inputs can be mapped from any stick switch or knob. ||

One input can be mapped to multiple output channels. ||

But to change a single output from multiple inputs, use 'Mixes'.||

Even the trims may be redefined here. || 

The trims inputs default to 1,2,3,4 and I suggest you don't change these unless you know what you're doing. It's possible to have a trim affect a different channel (up to 8) but this is rarely needed.

After making changes, Be SURE TO TEST on the ground before flying.||
