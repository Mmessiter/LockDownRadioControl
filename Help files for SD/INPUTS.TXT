This screen allows channels to be re-named and re-routed from any input and to any output.||

For each channel, the input number is displayed first (in yellow) then the channel name, then the channel output number (in red).||

All three of these items can be edited.

(1) A channel can be routed from any input device.|
(2) The name of each output channel can be changed.|
(3) A channel can be routed to any output channel.||

All channels by default are routed from the input of the same number (in yellow) to the output of the same number (in red).|
Tap a name or a  number to bring up its edit window.||

Inputs 9,10,11 and 12 (and switches) offer three positions, which can be edited from their curve.||

Inputs 13,14,15 and 16 don't really exists so they are always centred. Nonetheless these last four channels can be very useful for mixes and extra functions. Their inputs can be mapped from any stick switch or knob, and their can might do anything you choose.||

One input can be mapped to multiple output channels, but don't map an output from more than one input channel. Use a mix for this.||

But to change a single output from multiple inputs, use 'Mixes'.||

The trims' output channel may be redefined here. But trims' inputs cannot be changed because there are only the four hardware trims. || 

The trims outputs default to 1,2,3,4 and I suggest you don't change these unless you know what you're doing. |
It's possible to have a trim affect any channel but this is rarely needed.

After making changes, Be SURE TO TEST on the ground before flying.||
