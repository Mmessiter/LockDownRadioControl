This screen offers the option to use a third party transmitter module plugged in at the back, instead of the built-in transceiver. ||Using such a module allows the use of other receivers and protocols, such as Spektrum.||

The module must be of the 'JR' type.||

Features such as telemetry and model-match are not supported with most third party modules; so use the timer to avoid running out of battery and be sure the correct model memory is loaded before connecting.||

When not using a module, it's best to remove it so that it cannot consume power of transmit anything. Put the blank cover in its place.||

After changing the transmitter module settings, the transmitter must re-boot.
