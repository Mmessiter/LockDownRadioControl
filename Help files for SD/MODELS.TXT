Use the + and - buttons to select a model from 1 to 99.||

Re-name a model by clicking its name and typing a new one.||

'Backup ->' creates (or overwrites) a separate file of the current model.|

Before pressing Export, click on the green name and type a new one.
Use 8 or fewer characters. The system will add '.MOD'. ||

'<- Restore' reads a file into the currently selected model number, 
overwriting whatever was there before - so BEWARE!||

To 'Send' a model memory file wirelessly to another transmitter:||
(1) Put both transmitters onto the 'Models' screen. |
(2) Get the model file to be sent into view at the send end. |
(3) Press 'Receive' at the Receive end...|
(4) WITHIN 30 SECONDS press 'Send' at the Send end.| 

The transfer will create (or overwrite) the file at the receive end. ||

The received file will have the same name as the sent file.||

After sending a model memory, be sure to CHECK ALL FUNCTIONS before flying!!||

It's possible that some switches etc. may be reversed because of a difference in how each transmitter was built. |
Edit as needed before flying!
