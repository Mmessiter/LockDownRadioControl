This screen should usually be visible while flying a model because it shows the bank, time, trims, motor timer and battery voltages. ||

The motor timer runs automatically when the model is NOT in bank 4 in order to time the motor use. So it's recommended to set up the model such that the motor is always off in Bank 4.||

The timer pauses while in Bank 4 and is reset when the transmitter is turned on.||

LED blue = radio is not trying to connect.|
LED red = radio is trying to connect.|
LED green = connection exists.||

When the LED is green, the 'Off' button is disabled to prevent accidental disconnection. ||
If you want to switch off while still connected, you can from the Options screen.||

The connection quality indicator (near the bottom of the screen) shows the proportion of successful data packets as a percentage of total data packets averaged over a six second||
period. 

If the connection quality changes very quickly, the indicator will move over the next few seconds to reflect the change. Packets which did arrive at the receiver but whose 'acknowledgment' was lost are counted as failed.||    

The 'Off' button works immediately when not connected, but will need to be held down for a few seconds (definable) when connected for reasons of safety.||


Press 'Setup ...' to change all settings.
