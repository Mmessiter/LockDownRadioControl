This screen displays GPS data if a GPS device is connected via the 'Sensors Module'.||

The 'Zero alt' button marks the current altitude as zero. If not pressed, altitude is relative to sea level. So press this button while on the ground in order later to see the model's altitude above the ground where you are.||

The 'Mark here' button marks or remembers the current location. Thereafter, 'Meters to mark' and 'Heading to mark' and 'Max distance' are calculated relative to this marked location. If you mark your location before take off, this will therefore tell you how far you went and which way is back again.
