The Model Setup screen offers all options for setting up the model.||

The options for setting up the transmitter are to be seen on the Transmitter Setup screen.

When you connect to a new model that is unknown to the transmitter, the ID of that model is automatically stored in the current model memory. That's why the 'Unmatch' option exists: If you accidentally	 connected when the wrong model memory was loaded, this option allows that ID to be removed from the model memory.

To return to the Front screen, press 'OK'.
