Buddy-Boxing:||

This feature allows dual control which is probably essential when teaching beginners to fly.||

It needs two transmitters of course, one for the 'Master' and one for the pupil.||

The Master uses a switch on the Master's transmitter to give control to the Buddy - and to take control back fast when pupil screws up - as they usually do.||

When using the Buddy box feature, a switch on the Master transmitter must be defined as 'Buddy'. It's probably best if this is a spring-return switch because then simply releasing the switch will return full control to the Master instantly. ||

On the Master transmitter, you can select which channels are enabled for the Buddy. Any channels not selected will remain always under the control of the Master even after selecting Buddy with the switch.||

The same model memory should be loaded on both transmitters, and the Buddy (pupil) needs to know the ID of the Master transmitter. Both of these are easily achieved by sending the model memory from the Master to the pupil wirelessly. Embedded in the data sent is the needed ID of the sending transmitter. ||

If AMS (automatic model selection) is enabled on both transmitters and both transmitters have saved the ID of the model in use, then BOTH transmitters will automatically select the correct model if either had the wrong model loaded.|| 

When moving the Buddy switch, both transmitters will announce either 'Master' or 'Buddy' in order to make clearer who is in control.||

The wireless buddy feature is even supported when using a third party transmitter module. ||

Wired Buddy-Boxing:||

Wired buddy-boxing is also supported. On rare occasions this might be desirable if interference is suspected.||

Connect the two transmitters with a normal 3.5 mm jack audio cable, mono or stereo.||

The selected buddy channels are saved on a per model basis.||

After changing buddy-box settings, the transmitter should be re-started.||
