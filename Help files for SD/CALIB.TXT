It's VERY important to use this option before setting up and flying.||

If the transmitter is new, you should start by selecting Reset Transmitter. This function will take a few minutes to complete. It deletes all model memories but none of the backup files. It also sets sticks mode to mode 2, so remember to change this (under TX settings) if you fly mode 1.

Next calibrate the sticks and switches or knobs.

The system needs to know the end points and centre points of the sticks' and the knobs' potentiometers. Only channels 1-8 need to be calibrated.||

Press the button ONCE, then wiggle the sticks and move the knobs several times to all their extreme positions, then press the button ONCE again, then put all to their exact centre positions and move all four edge switches away from you before pressing it one last time to record centres switch orientation.||

On his screen the hardware trim directions are also defined, and there's an option to reset the transmitter. This might be needed if you want quickly to reset everything to defaults.
