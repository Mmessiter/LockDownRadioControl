While using this screen, transmission is stopped. 

Use the + and - buttons to select a model from 1 to 99.
Re-name a model by clicking its name and typing a new one.

'Export ->' creates or overwrites a separate file of the current model.
'<- Import' reads a file into the current model, overwriting it. (Beware!)

To 'Send' a model memory file to another transmitter, put both transmitters
onto the 'Models' screen. Get the model file to be sent into view at the 
send end. Press 'Receive' at the Receive end, and WITHIN 10 SECONDS
press 'Send' at the Send end. The transfer will create (or overwrite) the 
file at the receive end.
